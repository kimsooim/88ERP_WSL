## 기술 스택 결정
- Frontend: Next.js 14 (App Router), TypeScript, Tailwind CSS
- UI: shadcn/ui
- Backend: Supabase (PostgreSQL)
- AI: Claude API / OpenAI GPT
- Deployment: Vercel
- State: Zustand
- Forms: React Hook Form + Zod

## 주요 기능
1. 포트폴리오 갤러리 (OEM 제작 사례)
2. 실시간 견적 계산기
3. AI 상담 챗봇 (24시간 견적 상담)
4. 샘플 신청 시스템
5. B2B 관리자 대시보드
